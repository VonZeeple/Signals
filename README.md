Signals
=======

Signals is a **mod for the game Vintage Story** (a voxel-based survival game).

# How to Contribute

## General Philosophy

While modding is mostly about coding, it is also an artistic and creative endeavor—deeply personal. Please respect the vision and artistic direction of the project.

The goal of this mod is to encourage problem-solving skills by keeping the number of components to a minimum. New components should only be added if a function cannot be built with existing elements.

Future features are planned to make circuits smaller while maintaining complexity.

Feel free to share your ideas! If they don’t fit within the base mod’s scope, they might be implemented as an addon.

## Contributing Code

You can contribute by creating a merge request on GitHub.

## Contributing Assets

The process is the same as for code. Please do not use content under licenses incompatible with Apache v2.

## Reporting Bugs

Reporting bugs and odd behaviors is essential for the project’s growth. Please create an issue on GitHub: https://github.com/VonZeeple/Signals/issues

## About AI

- Using AI for *coding* is accepted, with these guidelines:
  - Generating full features from prompts (“vibe coding”) is fine for development branches or quickly testing new features. However, remember that code reviews and further development may be done by humans, so code must remain concise, readable, and maintainable. Avoid sloppy code.
  - Not everyone has access to advanced AI tools; assume code will be read and maintained by humans.
  - Help others improve AI-generated code by adding relevant skill files or persistent configuration documents (e.g., CLAUDE.md if you use Claude) to the repo.

- Using AI for *art* generation (models, images, sounds, etc.) is **not** allowed. Using AI to alter models for technical (non-creative) reasons (e.g., normalizing sizes) is acceptable.
